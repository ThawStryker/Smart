# Comet 中文本地化 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 9 个 Comet SKILL.md / 参考文档从英文翻译为简体中文，技术标识符保持原文。

**Architecture:** 逐文件翻译，每个文件自包含。先翻译主工作流和参考文档，再按 Phase 顺序翻译子命令，最后翻译预设路径。全部完成后做一致性检查。

**Tech Stack:** 纯 Markdown 翻译，无代码变更。

- [ ] **Step 2: 翻译为简体中文**

将正文内容翻译为中文。保留以下内容不变：

- phase → 阶段

- preset → 预设

- delta spec → 增量规格
- blocking point → 阻断点
